# Boas Práticas de Segurança - MikroTik

## 🔐 Medidas aplicadas

- Restrição de acesso ao Winbox apenas via rede interna
- Uso de VPN para acesso remoto seguro
- Bloqueio de portas desnecessárias
- Controle de conexões inválidas
- Segmentação básica de rede

## ⚠️ Riscos mitigados

- Acesso não autorizado
- Exposição de serviços na internet
- Ataques de força bruta
- Exploração de vulnerabilidades conhecidas

## 📌 Observação

A segurança em dispositivos MikroTik depende diretamente da correta configuração de firewall, controle de acesso e atualização constante do sistema.
